# javascript-racingcar-precourse

### 기능 명세서

- [ ] 사용자에게 경주할 자동차들을 입력 받을 수 있다.

  - [ ] 각 자동차에 이름 부여 가능
  - [ ] 자동차 이름이 5자가 넘어가면 `[ERROR]`
  - [ ] 자동차 이름은 쉼표(`,`)로 구분

- [ ] 사용자에게 이동 할 횟수를 입력 받을 수 있다.

  - [ ] 숫자가 아닌 값이 입력된다면 `[ERROR]`

- [ ] 자동차는 전진할 수 있다.
  - [ ] 무작위 숫자가 4이상일때
- [ ] 자동차는 멈출 수 있다.

  - [ ] 무작위 숫자가 4미만일때

- [ ] 우승자를 출력할 수 있다.
  - [ ] 여러명일 경우 쉼표(`,`)로 구분
